Proyectos

A lo largo de mi carrera, he colaborado en diversos proyectos de desarrollo web y móvil, aplicando metodologías ágiles y herramientas modernas para entregar soluciones de alto impacto. Algunos ejemplos son:

1. Banca Móvil Fassil (Versión Híbrida)

  • Tecnologías: Ionic / Angular, Ruby on Rails, APIs REST, Firebase
  • Descripción: Migración de la app de banca móvil nativa a un enfoque híbrido, permitiendo mayor velocidad de desarrollo y un mantenimiento más sencillo. Incluyó manejo de notificaciones push, validaciones de seguridad y sincronización de datos en tiempo real.
  • Rol en el proyecto: Líder técnico y desarrollador backend, coordinando a un equipo de 5 personas.

2. Sistema de Inmobiliarias Multi-Plataforma

  • Tecnologías: Ruby on Rails (RoR), Vue.js, PostgreSQL
  • Descripción: Plataforma para gestión de propiedades y clientes en línea, incluyendo panel administrativo, portal de usuarios y un módulo de notificaciones por correo/SMS. Múltiples roles de usuario y filtros de búsqueda avanzada.
  • Rol en el proyecto: Arquitecto de software y desarrollador full stack, implementando la estructura de base de datos y la integración del frontend con el backend.

3. Chatbot Conversacional para Atención al Cliente

  • Tecnologías: Python (Flask, Django), NLP (Natural Language Processing), Integración con Messenger y WhatsApp
  • Descripción: Desarrollo de un chatbot que responde de manera automática consultas de clientes, gestionando datos y solicitudes comunes en el sector financiero. Incluyó análisis de lenguaje natural y reconocimiento de intenciones.
  • Rol en el proyecto: Encargado del backend y de la configuración de los servicios de mensajería. Contribuí en la implementación de modelos de NLP y la lógica de negocios.

4. Plataforma de Gestión de Eventos (Ticketing)

  • Tecnologías: Ruby on Rails, Android Studio, API REST, Payment Gateway
  • Descripción: Sistema de venta de entradas en línea con app móvil para escaneo de QR en el ingreso a eventos masivos. Soporta alto número de transacciones y validaciones en tiempo real.
  • Rol en el proyecto: Diseñé la arquitectura de software y lideré el desarrollo móvil, integrando un gateway de pago para transacciones seguras.
